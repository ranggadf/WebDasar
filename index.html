Web Kelompok 4

sebelum membuat CSS, tampilan web masih polos. dengan modifikasi demikian membuat tampilan web lebih bervariasi. CSS *{...} digunakan untuk mengatur model kotak CSS untuk semmua elemen dalam halaman web. CSS header{...} digunakan untuk mengatur area judul/header. CSS nav{..} dan nav ul{...} digunakan untuk mengatur area navigasi dan model list navigasi. CSS article{...} digunakan untuk mengatur area articel. CSS section::after{...} digunakan untuk mengatasi masalah float di dalam elemen HTML. CSS @media (max-width: 600px){...} digunakan untuk menentukan gaya tertentu yang akan diterapkan pada halaman web berdasarkan karakteristik perangkat yang sedang digunakan oleh pengguna
